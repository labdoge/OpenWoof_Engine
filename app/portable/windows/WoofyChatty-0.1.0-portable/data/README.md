# Portable Data Folder

## 繁體中文

這個資料夾會由 portable app 寫入本機資料、設定、vault、log 與 WebView2 user data。發布 zip 時可以保留空資料夾，但不要放入任何實際使用者資料、API key、vault 或 log。

## English

This folder is where the portable app writes local data, settings, vault files, logs, and WebView2 user data. Keep the empty folder in the release zip, but never include real user data, API keys, vault files, or logs.
